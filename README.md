# Objectif

Le but des documentations suivantes est d'expliquer de manière la plus complète possible comment fonctionnent les NAS et d'essayer de faciliter les choix à faire pour construire son propre NAS.

# Les documentations

|                         Nom                         |                                                                     Objectif                                                                      |
| :-------------------------------------------------: | :-----------------------------------------------------------------------------------------------------------------------------------------------: |
|            [C'est quoi un NAS ?](NAS.md)            |                                    Comprendre ce qu'est un NAS et dans quelles situations ça peut être utile                                      |
|   [Pourquoi faire son propre NAS ?](selfhost.md)    | Une documentation qui compare des exemples de NAS personnalisés avec des marques connues et les cas d'usage de chacun pour aider à faire un choix |
| [Les différentes solutions de stockage](storage.md) |                          Comparaison entre ZFS, XFS et BTRFS, explication des types de RAID (des RAIDZ pour être précis)                          |
|             [Les différents OS](OS.md)              |                 Une liste des différents OS majoritairement utilisés pour les NAS self-host avec leurs avantages et inconvénients                 |
|    [Les composants à privilégier](composants.md)    |                              Explication des composants nécessaires, des composants à privilégier et pourquoi, etc.                               |
|       [Quelques exemples de build](builds.md)       |                                                C'est dans le nom, faut pas être trop con non plus                                                 |

<span style="font-size: smaller;">Il est préférable de lire les documentations dans l'ordre afin d'être sûr de comprendre tous les points abordés.</span>
